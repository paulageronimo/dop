Here's how to set it up!

01
Register Students

This app uses the Howdy App API & Canvas API to provide information about student class locations. However a List of UINs is Required

02
Request your Students Register

Students must either download the app or fill out the survey on the website in order to get notifications.

03
Report

If a professor or a student reports/self-reports they have a lethal virus, the app will automatically notify all students & faculty in all F2F classes of possible exposure to a lethal virus.
